# cc-connect

Bridge local AI coding agents (Claude Code, Cursor, Gemini CLI, Codex) to messaging platforms (Feishu/Lark, DingTalk, Slack, Telegram, Discord, LINE, WeChat Work).

Chat with your AI dev assistant from anywhere.

This npm package is the ChatArch distribution of `cc-connect`. It installs the `cc-connect` CLI by downloading the matching binary from the ChatArch GitHub Release.

## Install

```bash
npm install -g @chatarch/cc-connect
```

## Usage

```bash
# Create config
cc-connect --version

# Edit config.toml, then run
cc-connect
cc-connect -config /path/to/config.toml
```
